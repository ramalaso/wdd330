Forms are made up of a form element that contains form controls such as input fields, select menus and buttons.
Each form control has an initial value that can be specified in the HTML code.
The legacy DOM had a useful property called document.forms that returns an HTML collection of all the forms in the document.
Form objects have a number of useful properties and methods that can be used to interact with the form.
Forms trigger a number of events like those discussed in the last chapter. Some of these events are exclusive to forms.
Other types of form controls: Input Fields, Text Input Fields, Password Input Fields, Checkbox Input Fields, Radio Button Input Fields, Hidden Input Fields, File Input Fields, Other Input Types.
Object-oriented programming is often used to model representations of objects in the real world. There are three main concepts in OOP: encapsulation, polymorphism and inheritance.
